# Governance

## Project Lead

AgentGit is maintained by **Christopher Anderson** ([@stand-sure](https://github.com/stand-sure)), who serves as Benevolent Dictator for Life (BDFL) for the project.

## Decision Making

- **Bug fixes and minor improvements:** Merged at maintainer discretion after CI passes
- **New features:** Discussed in GitHub Issues or Discussions before implementation
- **Breaking changes or architecture decisions:** RFC posted in GitHub Discussions with at least 1 week for community feedback

## Contributions

All contributions are welcome via pull request. See [CONTRIBUTING.md](CONTRIBUTING.md) for guidelines.

Contributors do not need to sign a CLA. The MIT license ensures all contributions are available under the same terms.

## Succession Plan

If the project lead becomes unable to maintain the project:

1. Active contributors with merged PRs may request maintainer access
2. The project may be transferred to a community organization
3. The MIT license ensures the code remains freely available regardless of maintenance status

## Communication

- **Bug reports and feature requests:** [GitHub Issues](https://github.com/innago-property-management/AgentGit/issues)
- **Questions and ideas:** [GitHub Discussions](https://github.com/innago-property-management/AgentGit/discussions)
- **Security vulnerabilities:** See [SECURITY.md](SECURITY.md)
